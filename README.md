# 🧠 Bootcamp de Engenharia de IA — triggo.ai

Repositório de atividades e estudos desenvolvidos ao longo do **Bootcamp de Engenharia de IA da triggo.ai** — programa intensivo de 6 semanas focado em agentes inteligentes, arquitetura de soluções e desenvolvimento orientado por IA.

---

## Sobre este repositório

Este repositório documenta minha jornada prática durante o bootcamp: exercícios, experimentos, anotações técnicas e reflexões sobre cada trilha de aprendizado. O objetivo não é apenas registrar o que foi feito, mas consolidar raciocínio aplicado — o que funcionou, o que não funcionou, e por quê.

> *"Mais do que ensinar tecnologia, queremos desenvolver a forma de pensar de quem vai atuar com IA no mercado."* — triggo.ai

---

## Stack e Ferramentas

As tecnologias utilizadas nas atividades acompanham o que é trabalhado nas trilhas do bootcamp:

- **Linguagens:** Python
- **Frameworks de IA:** LangChain · LangGraph · OpenAI SDK
- **Agentes & Orquestração:** LangGraph Agents · Tool calling · ReAct pattern
- **Arquitetura:** RAG · Vector stores · API design
- **Boas práticas:** Clean code · Estrutura modular · Documentação técnica

> Stack complementar que trago de projetos anteriores: FastAPI · FastMCP · ChromaDB · Groq · React Native/Expo.

## Como navegar

Cada pasta de semana contém:

- **`exercicios/`** — código das atividades, com comentários explicando as decisões de implementação
- **`notas.md`** — registro de aprendizados, dificuldades encontradas e conexões com problemas reais
- **`README.md`** — contexto da semana: o que foi abordado, o que foi entregue, principais insights

A ideia é que qualquer pessoa consiga entender não só *o que* foi feito, mas *como e por que* foi feito dessa forma.

## Contexto e Motivação

Tenho background em Análise e Desenvolvimento de Sistemas (CESAR School, 2026) e experiência prévia com engenharia de agentes de IA — incluindo a construção de um servidor MCP com LangGraph integrado a uma pipeline de dados em ambiente de produção.

Entrei neste bootcamp com interesse específico em aprofundar boas práticas de arquitetura para sistemas agênticos, especialmente em contextos corporativos onde confiabilidade, rastreabilidade e segurança importam tanto quanto a funcionalidade.
